## HouZEN
Um aplicativo para pessoas buscarem espaço e também ofertarem espaço disponível em seu ap ou casa para dividir com outra pessoa.

## Tecnologias 

As tecnologias utilizadas nesse projeto foram:

* Javascript
* React Native
